cocktail-proxy-java

Available GET endpoints:

/random-drink - fetches a random drink and the name celebrated today

/proxy/filter.php?i=vodka - acts as a proxy for the TheCocktailDB API

/local-db - displays all drinks fetched from TheCocktailDB so far

/drink-of-the-day/2023/12/24 - displays the drink associated with the specified date (/yyyy/mm/dd)
